## Nutri-Pal's website

this is the repository for nutri's website 

'fork it' or 'clone it'
then run

###### // clone it
```terminal
$ git clone https://github.com/nutri-pal/nutripal-website.git 
```

###### // go to app
```terminal
$ cd nutrial-website 
```

###### // install packages
```terminal
$ yarn or yarn install or npm install
```

###### // run the app
```terminal
$ yarn start or npm start
```
